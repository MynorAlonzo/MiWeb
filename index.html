HTML

Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto.

CSS

La sigla CSS corresponde a la expresión inglesa Cascading StyleSheets, que puede traducirse como «Hojas de estilo en cascada». El concepto se utiliza en el ámbito de la informática para referirse a un lenguaje empleado en el diseño gráfico.

JavaScript

Es el nombre de un lenguaje de programación: es decir, un lenguaje formal que brinda instrucciones a una computadora (ordenador) para generar ciertos datos. Se utiliza sobre todo para producir recursos interactivos en una página web.
